About This Project

The Artificial Intelligence (AI)

This system uses a Convolutional Neural Network (CNN) to classify injury images. Specifically, it uses a pre-trained model called MobileNetV2 with a new classification head added on top (a technique called Transfer Learning).

  • The model was trained on a dataset of images sorted into classes (like 'burns', 'wounds', etc.).
  • When you upload an image, it is resized to 224x224 pixels and normalized.
  • It's then passed through the injury_classifier.h5 model, which outputs a confidence score for each possible injury type.
  • The highest score is chosen as the final prediction.

The Database (DBMS) Component

This web page demonstrates the AI model but does not connect to the database for security reasons. The full project, as shown in your database.py script, connects to a MySQL database named injury_ai.

  • The predictions table is designed to log every analysis performed by the system.
  • It stores the predicted injury_type, the confidence score, the image_path, and a timestamp.
  • This database creates a valuable, persistent record of the model's performance and usage, which can be used for auditing, further model refinement, or tracking injury trends.
